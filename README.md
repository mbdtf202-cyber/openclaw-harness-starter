# OpenClaw Harness Starter

Starter repo for building reusable OpenClaw harnesses.

This project is generated by `openclaw harness init` and is designed for:

- message scenario harnesses
- channel contract tests
- CLI output harnesses
- plugin test kits that depend on `openclaw/plugin-sdk/test-utils`

## Quick start

```bash
pnpm install
pnpm test
```

## What is included

- `src/message-scenario.test.ts`
  A ready-to-run example using `createMessageScenarioHarness`
- `src/cli-output.test.ts`
  A minimal stdout/stderr capture example using `captureProcessWrites`
- `.github/workflows/ci.yml`
  GitHub Actions test workflow

## Public OpenClaw APIs used here

- `openclaw/plugin-sdk/test-utils`
  - `createMessageScenarioHarness`
  - `createTestRegistry`
  - `createOutboundTestPlugin`
  - `captureProcessWrites`

## Typical next step

Replace the demo tests with one of these:

1. your plugin outbound contract tests
2. your channel routing regression tests
3. your CLI output and error-path tests

## Upgrade policy

This starter pins `openclaw` to `^2026.3.3`.

When you want newer harness helpers:

```bash
pnpm up openclaw
```
